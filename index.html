N-Puzzle Game

C# Programming & Design

This Program is designed to replicate an N-Puzzle Game. This is a sliding puzzle that consists of numbered square tiles in random order with one tile missing. The N-Puzzle Game can also be generated at any size. But there is a Restriction in that the game can only be generated to a size in that the column and row of the game must be the same size. So for example 3x3 or 4x4. The game also allows for the player to SAVE a current game and also OPEN a previously played game.

  • Date: 2015
  • Client: Learning Project
  • Category: Programming & Design

Tic-Tac-Toe

C# Programming & Design

This Programs is designed to replicate an Tic Tac Toe Game. The Tic Tac Toe game requires two player, one that represents ‘X’ and another that represents ‘O’. The players have the option to select any of the 9 locations on the game board, but the player who plays first will always be player ‘X’. The game will determine which player wins by whether ‘X’ or ‘O’ is the first to form a pattern in the game.

  • Date: 2016
  • Client: Learning Project
  • Category: Programming & Design

Travel Booking Program

C# Programming & Design

Program is designed to reserve seats for passengers who use a travel agency “Do-It All Travels”. The program is designed to hold 15 seats for passengers and 5 waiting list spots for additional passengers. Passengers can be added, deleted or put on a waiting list. The program is sufficiently designed to organize and relocate passengers whenever appropriate. Error check is also implemented into the program
